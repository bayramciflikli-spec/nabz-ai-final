# NABZ-AI – Portlar (kalıcı)

## Port 3000 – Next.js dev

Bu projede **dev sunucusu her zaman port 3000** üzerinde çalışır.

- **Adres:** http://127.0.0.1:3000 veya http://localhost:3000
- **Script'ler:** `npm run dev`, `npm run dev:127` → `--port 3000`
- **Batch:** NABZ-AC.bat, NABZ-Masaustu.bat → 3000'i kapatır, 3000'de başlatır
- **Port kapatma:** 3000-kapat.bat

Başka port kullanma; tüm referanslar 3000'e sabitlendi.

## Port 5050 – Python Flask (NABZ yerel API)

Yerel entegrasyon sunucusu **port 5050** üzerinde çalışır (Chrome uyumlu).

- **Adres:** http://127.0.0.1:5050
- **Endpoint:** POST /islem → NABZ API'ye iletir
- **Başlatma:** NABZ-Flask.bat veya `python scripts/nabz_local_server.py`
- **Port kapatma:** 5050-kapat.bat
- **Detay:** scripts/README.md
